# Web Dev 2 – Starter (JS)

Starter-level Next.js app to post/join rides for campus events.
No auth, no DB, no external APIs — just localStorage + a static `events.json`.

## Quick start
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Structure
- `app/` App Router pages
- `components/` React components
- `data/events.json` Demo events
- `app/globals.css` Tailwind base
- `tailwind.config.js`, `postcss.config.js`, `jsconfig.json`

## Build & run
```bash
npm run build
npm start
```
Deploy to Vercel by importing the repo.
